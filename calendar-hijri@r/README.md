# Calendar with Hijri

A fork of the Cinnamon calendar applet with an optional Islamic (Hijri) date display. Uses the offline Kuwaiti algorithm — no internet required.

## Features

- All original Cinnamon calendar functionality (events, week numbers, custom date formats)
- Optional Hijri date shown below the Gregorian date in the calendar popup
- Language follows system locale (supports English, Русский, العربية, Deutsch, Français, Español, Türkçe, Bahasa Indonesia, فارسی, اردو and more)
- Offline Tabular Islamic calendar algorithm — no API calls

## Install

Open **Applets** → **Download** tab → search for "Calendar with Hijri".

## Usage

Right-click the applet → **Configure** → enable **Show Islamic (Hijri) date**.

The Hijri date appears in the calendar popup header below the Gregorian date.
