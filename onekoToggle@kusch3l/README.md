# Oneko Toggle

An applet to toggle everyones favourite display cat.

## Features

- Toggle oneko on click on applet
- Update icon of applet
- Script included to add to your autostart

## Requirements

Oneko installed and working (sudo apt install oneko)

## Installation

Add applet to panel of your liking. Left click on applet. If it is not working, you need to allow the oneko.sh script to run as programm (chmod +x ~/.local/share/cinnamon/applets/onekoToggle@kusch3l/oneko.sh)
