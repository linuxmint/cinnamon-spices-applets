Mode adoption d'applet — version clic prioritaire

1. Installer/activer c-eyes normalement sur le panneau.
2. Placer c-eyes juste à gauche de Créature.
3. Dans Créature > Animation, choisir “Adopter l’applet juste avant”.
4. Laisser “Clic prioritaire sur Créature” activé.

L’œil reste visible derrière, mais le clic est capturé par une couche transparente de Créature.
Pour cliquer directement sur c-eyes, désactiver “Clic prioritaire sur Créature”.
