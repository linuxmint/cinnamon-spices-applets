# Fade Monitors Applet

Tray icon applet for the Fade Monitors enhanced dimming script https://github.com/hisovereign/Fade-Monitors/blob/Fade-Monitors-enhanced-dimming/README.md

Toggle for mouse-based and idle dim, buttons to start and stop script, and buttons for user to open script and config folder.

**Requirements:**
- Fade Monitors enhanced-dimming script installed in /.local/bin/
- X11 session
