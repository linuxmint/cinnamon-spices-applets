# RSSDock for Cinnamon
A lightweight, multi-source RSS news ticker for the Cinnamon Desktop environment. Optimized for modern hardware like the Lenovo LOQ series.

## Features
- **Smooth Scrolling:** CPU-friendly manual scrolling engine.
- **Multi-Source:** Add as many RSS feeds as you want.
- **Time Stamps:** See exactly when the news was published [HH:MM].
- **Custom Refresh:** Set update intervals and refresh manually from the menu.
- **Tooltips:** Hover over truncated titles to see the full headline.

## Installation
1. Download the `rssdock@yourid` folder.
2. Copy it to `~/.local/share/cinnamon/applets/`.
3. Enable it from the Cinnamon Applets settings.
