# &#65279; ![CinnVIIStarkMenu Logo](https://git.io/vDWaq) CinnVIIStarkMenu

### Features:

 * Windows 7 start menu or MATE menu layouts
 * Various other configuration options.
 * Configurable quicklaunchers
 * Keyboard navigation
 * Compatible with all Cinnamon versions since 2.6
