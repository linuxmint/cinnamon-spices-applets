# Asset licensing

## Applet icon

`icon.png` and `6.0/icon.png` contain the same original generic white menu
mark created specifically for Menu Eleven Reloaded. They are distributed
under GPL-3.0-only with this package.

## Desktop-theme icons

Search, calculator, and Wikipedia result icons are not bundled. At runtime,
the applet requests generic symbolic icons from the user's installed desktop
icon theme.

## Removed assets

No Google, Bing, Yahoo, DuckDuckGo, Ask, Ecosia, Startpage, Brave, Qwant,
Wikipedia, Microsoft, or other company logo is included in the package.
