## Cinnamon Theme Editor

Create your own Cinnamon theme. The theme will be saved as "custom-theme" in your `~/.themes/` directory.

### Installation

The easiest way to install an applet is to right click on the panel you want to add the applet to and select "Applets". Then go to the "Download" tab to download the applet and then add it to the panel on the "Manage" tab.

### Dependancy

This applet requires `sassc` to be installed.

For debian based systems: `sudo apt install sassc`.<BR>
For arch based systems: `sudo pacman -S sassc`.<BR>
For fedora based systems: `sudo dnf install sassc`
