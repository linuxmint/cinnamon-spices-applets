# Window Closer

Quickly close multiple windows from one place.

Click the panel icon to open a fullscreen overlay showing all open windows as cards
with live thumbnails. Click any card to close that window. The overlay stays open so
you can close several windows in a row. Press Esc or click the backdrop to exit.

## Features

- All open windows displayed as uniform cards in a clean grid
- Live window thumbnails
- One click anywhere on a card closes that window
- Works across all workspaces
- Keyboard support (Esc to exit)
