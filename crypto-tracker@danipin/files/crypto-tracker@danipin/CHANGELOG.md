# Changelog

All notable changes to the "Crypto Tracker" applet will be documented in this file.

## [1.1.0] - 2026-02-17

### Added
- **Portfolio Tracker:** New view to manage crypto holdings, calculate total balance, and track profit/loss.
- **Alarm Management:** Centralized hub for active and past alarms, including volatility alerts and an archive.
- **API Statistics:** Detailed breakdown of API usage (Basic vs. Key calls) to help manage monthly quotas.
- **Seasonal Themes:** Added Easter, Bitcoin Pizza Day, and Valentine's Day themes with animations.
- **Search:** Global search functionality to quickly find and add coins to the dashboard or portfolio.
- **Error Logging:** Robust error handling with user notifications and log file management.
- **Candle Charts:** Option to switch between Line and Candlestick charts in the detail view.
- **Compact Mode:** Space-saving display option for the panel.

### Changed
- **API Optimization:** Price updates now default to Basic calls (free tier) to save API Key quota for charts and details.
- **UI Improvements:** Unified hover effects, better layout for dropdowns, and smoother animations.
- **Translations:** Updated German translations and fixed missing strings (Vol:, Target, Current).
- **Settings:** Reorganized settings into logical sections (Display, Source, Alarms, Color).
