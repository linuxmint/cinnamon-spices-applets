# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-05-19

### Added
- New metric to display **GPU temperature** (supports dedicated and integrated GPUs).
- Support for numerous new **language packs** (e.g., Arabic, Bulgarian, Czech, Hindi, Indonesian, Russian, Polish).

### Improved
- Optimized **UI design** (improved spacing/padding in the panel, consistent icon alignment).

### Fixed
- Various **bug fixes** in the language files (resolved multiple definitions that caused crashes).
- Corrected profile notifications in the panel (fixed translation errors involving template literals).
