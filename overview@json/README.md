# Overview Dashboard

A sleek, macOS Launchpad-style applet for Cinnamon. This applet strikes the perfect balance between modern minimalism and traditional desktop utility, seamlessly blending clean aesthetics with rich functionality.

## Key Features

* **A Bridge Between Eras:** Gives a refreshing, modern lift to the Cinnamon desktop while retaining a familiar, nostalgic workflow.
* **Full Linux Mint Compatibility:** Fully optimized for Linux Mint 22.3 and its native Cinnamon desktop environment.
* **Curated Content Feed at Startup:** Displays the latest headlines from trusted news sources alongside curated updates covering movies, science, and philosophy right when you boot up.
* **Flexible Launching:** Trigger it instantly with a simple click on its panel icon, or bind it to any Cinnamon Hot Corner directly from the applet settings for gesture-based access.
* **Smart Weather Tracking:** Stay updated at a glance. Customize your location on the fly by typing your city name directly into the applet settings.
* **Dual Design Philosophy:** Crafted for users who want a distraction-free environment without sacrificing extensive customization and deep features.
